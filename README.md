AngoLink lda

Conversor de imagem para facilitar a edição de imagens no acto de inserção dos documentos do visto na plataforma:

https://www.smevisa.gov.ao/

Desenvolvi este programa devido a necessitadade borucratica das pessoas que estão a dar entrada de visto no nosso portal da sme que constantemente da problema devido a especificações dos ficheiros 

Este programa converte imagens e pdf para as especificaçoes extatas do sme que são:


Passport with at least 6 months validity: Passaporte com pelo menos 6 meses de validade

A copy of the passport page showing data and photo: Cópia da página do passaporte que mostra os dados e a fotografia

Maximum file size - 70 KB: Tamanho máximo do arquivo - 70 KB

Image format - JPEG: Formato da imagem - JPEG

Photograph with white background: Fotografia com fundo branco

One standard size photo: Uma foto de tamanho padrão

Width (381-571) pixels: Largura de (381 a 571) pixels

Height (496 - 744) pixels: Altura de (496 a 744) pixels

Maximum file size - 200 KB: Tamanho máximo do arquivo - 200 KB

Image format - JPEG: Formato da imagem - JPEG

Vaccination certificate: Certificado de vacinação

Valid international vaccination certificate: Certificado internacional de vacinação válido

Booking an air ticket: Reserva de passagem aérea

Booking of round trip air ticket: Reserva de passagem aérea de ida e volta
